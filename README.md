# 💰 Personal Finance Tracker

A command-line Personal Finance Tracker built with **Python** to help users record and manage their daily income and expenses. The application stores transaction data in a CSV file, provides summaries for a selected date range, and visualizes financial data using graphs.

> **Note:** This project was created while revising Python concepts by following a YouTube tutorial and implementing the project for practice.

## Features

* ➕ Add income and expense transactions
* 📅 View transactions within a selected date range
* 📊 Display total income, total expenses, and net savings
* 📈 Visualize daily income and expense trends using Matplotlib
* 💾 Automatically create and store data in a CSV file

## Technologies Used

* Python
* Pandas
* Matplotlib
* CSV Module
* Datetime Module
